### Assets

This directory contains static assets used by the lambdawaker package.

#### Contents

- `img/`: Image assets organized by category.
    - `country_shapes/`: Silhouette shapes of various countries.
    - `logo_bank/`: Banking related logos.
    - `logo_card_type/`: Logos for different card types (e.g., Visa, Mastercard).
    - `logo_stencil/`: Stencil-style logos.
- `text/`: Text-based assets for data generation.
    - `address/`: Street names and address components.
    - `first_name/`: Lists of first names.
    - `last_name/`: Lists of last names.
    - `institutions/`: Names of various institutions.
