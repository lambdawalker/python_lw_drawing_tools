### Draw

The `draw` module is the core component for generating graphics and visual patterns. It contains a wide variety of tools for creating complex backgrounds, shapes, and effects.

#### Submodules

- `card_background/`: Specialized generators for card-like backgrounds.
- `color/`: Tools for color generation, manipulation (including HSLuv support), and visualization.
- `gradient/`: Various gradient generators, including linear, radial, and voronoi-based gradients.
- `grid/`: Grid pattern generators.
- `header/`: Components for drawing headers or banners.
- `lines/`: A collection of line-drawing tools (parallel, wavy, sawtooth, square wave).
- `shaders/`: Implementations of shader-like effects.
- `shadow/`: Tools for adding shadows to elements.
- `shapes/`: Basic and complex shape generators.
- `svg/`: Utilities for working with SVG elements.
