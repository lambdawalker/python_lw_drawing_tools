### File

The `file` module provides utilities for file system operations and path management.

#### Submodules

- `operations/`: Basic file operations.
    - `write.py`: Utilities for writing data to files.
- `path/`: Path management and resolution.
    - `PathResolver.py`: A flexible tool for resolving paths relative to various roots.
    - `ensure_directory.py`: Ensures that a directory exists, creating it if necessary.
    - `wd.py`: Utilities for working with the current directory or project root.
