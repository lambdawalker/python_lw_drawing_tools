### Generate

The `generate` module contains various data generators for use in synthetic data creation.

#### Components

- `field/`: Specific generators for common data fields.
    - `address.py`: Street and address generation.
    - `firs_name.py`: First name generation.
    - `last_name.py`: Last name generation.
    - `institutions.py`: Institution name generation.
- `numbers.py`: Generators for various numeric formats.
- `pseudo_text_generator.py`: Generates text that resembles real language but is synthetic.
- `time.py`: Generators for date and time information.
